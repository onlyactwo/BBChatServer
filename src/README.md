玩法介绍：
    在“德州扑克”游戏中，你将拿到2张只有你能使用的手牌。

    如果所有的下注回合结束，到了最后摊牌的时刻
    玩家就会亮出他们的牌，拥有组合后最大的5张牌的玩家将赢得底池里所有的筹码。
